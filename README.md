Projeto SQL - Restaurante

Projeto de banco de dados em SQL para um restaurante fictício.

Objetivo

Criar a estrutura inicial do banco de dados `restaurante`, contendo tabelas para funcionários, clientes, produtos, pedidos e informações adicionais dos produtos.

Tabelas criadas

- funcionarios
- clientes
- produtos
- pedidos
- info_produtos

Ferramentas utilizadas

- MySQL Workbench
- SQL
- GitHub
